# 💼Tableau-Human-Resources-Analytics-Dashboard
End-to-end Tableau HR analytics project. Features workforce KPI tracking, termination analysis, interactive navigation, and a modern dark-themed UI for data-driven human resources management.

# 📝 Business Problem
HR managers struggled with fragmented data, making it difficult to visualize the current workforce composition, track termination trends, or analyze the relationship between employee demographics and compensation.

# 🎯Goal
To build a comprehensive dual-view dashboard system: a Summary Dashboard for high-level KPIs and trends, and a Detailed Dashboard for granular employee record analysis, featuring a modern, dark-themed UI.

# ⚡️Key Visuals & Storytelling
KPI Bans: Provided an immediate pulse check on Total, Active, and Terminated employees.

Trend Analysis: Utilized line charts to track historical hiring and termination patterns.

Interactive Tooltips: Engineered to provide instant context, including drill-downs into department roles and average salary/age metrics without cluttering the main view.

Collapsible Filter Menu: Implemented a UI pattern using floating containers to keep the interface clean while providing robust filtering capabilities.

Custom Navigation: Integrated a seamless navigation system to toggle between the Summary and Details pages.

# 📈 Business Impact & Insights
Workforce Visibility: Leaders can now immediately identify department-level turnover and headcount distribution.

Compensation Analysis: Correlates age and seniority with salary, helping to identify pay equity or distribution gaps.

User-Centric Design: By utilizing a mockup-first approach and Figma for background design, the dashboard ensures a non-technical audience can navigate complex workforce data intuitively.

# 🛠 Tech Stack
Visualization: Tableau Desktop / Tableau Public
Design & Prototyping: Figma, Draw.io
Data Processing: Python (Faker library for dataset generation), Tableau Data Modeling

# 📊 Data Source
Origin: Dataset generated using Python (Faker) to simulate realistic HR records, including hire dates, termination status, department hierarchies, and salary information.
Processing: Performed data cleaning, mapped correct data types, created hierarchical relationships (Department > Job Title), and developed complex calculated fields for active vs. terminated status and performance metrics.
